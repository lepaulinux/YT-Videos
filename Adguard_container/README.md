🚀 **Como instalar e configurar o AdGuard em LXC com Docker + teste no Fedora!**

Neste vídeo eu mostro passo a passo como instalar o **AdGuard Home** dentro de um container **LXC**, executando via **Docker**. Além da instalação, você vai ver toda a configuração que eu utilizo para deixar o DNS mais seguro, rápido e com bloqueio eficiente de anúncios, rastreadores e domínios maliciosos.

No final, faço uma demonstração prática utilizando o **Fedora**, configurando o sistema para usar o IP do AdGuard como DNS padrão — assim você pode ver o funcionamento real na rede.
