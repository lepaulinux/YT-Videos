# 📂 Repositório de Scripts e Tutoriais

Neste repositório você encontra **scripts, comandos e tutoriais** utilizados nos meus vídeos publicados no YouTube.

O objetivo deste repositório é servir como **material de apoio prático**, facilitando o acompanhamento dos vídeos e permitindo que você **reproduza, estude e adapte** os exemplos apresentados, especialmente nas áreas de:

- Linux 
- Docker e containers 
- Proxmox e virtualização 
- Homelab e ambientes self-hosted 
- Automação e administração de sistemas 

Todo o conteúdo aqui é focado em **aprendizado prático**, do básico ao intermediário, sempre com exemplos reais usados nos vídeos do canal.
