# Português

Faça o download do arquivo `setup.sh` pelo site do GitHub.

Ou utilize o comando abaixo:

```bash
wget https://raw.githubusercontent.com/lepaulinux/YT-Videos/refs/heads/main/LXC_Script_Setup/setup.sh
```

Não se esqueça de tornar o script executável:

```bash
chmod +x setup.sh
```

Para executar o script, utilize um dos comandos abaixo:

```bash
./setup.sh
```

ou

```bash
bash setup.sh
```

# English

Download the `setup.sh` file from the GitHub website.

Or use the command below:

```bash
wget https://raw.githubusercontent.com/lepaulinux/YT-Videos/refs/heads/main/LXC_Script_Setup/setup.sh
```

Don't forget to make the script executable:

```bash
chmod +x setup.sh
```

To run the script, use one of the following commands:

```bash
./setup.sh
```

or

```bash
bash setup.sh
```
